# Leafly 🌿 – Sua Biblioteca Digital Pessoal

O **Leafly** é uma aplicação web moderna e responsiva para quem deseja **organizar, catalogar e acompanhar o progresso de leitura de seus livros**. A plataforma oferece estatísticas interativas, busca inteligente e uma biblioteca visualmente agradável, ideal para leitores que querem transformar sua coleção em algo prático e inspirador.

---

## ✨ Principais Recursos

- Dashboard intuitivo com estatísticas de leitura: total de livros, progresso, finalizados e páginas lidas.
- Biblioteca em cards: exibe capas, títulos, autores e informações rápidas.
- Busca e filtros para encontrar facilmente livros por título, autor ou gênero.
- Design responsivo e moderno, adaptado para desktop, tablet e celular.

## ⚙️ Funcionalidades

### Dashboard Principal

- 📊 Exibe estatísticas gerais da biblioteca:  
  - Total de livros cadastrados  
  - Livros sendo lidos  
  - Livros finalizados  
  - Total de páginas lidas  
- 🚀 Navegação rápida para outras seções  
- 📱 Design responsivo e atrativo  

### Biblioteca (Listagem de Livros)

- 📚 Mostra todos os livros cadastrados em formato de **cards**  
- 🔍 Sistema de **busca por título ou autor**  
- 🎭 Filtros por **gênero literário**

## 🛠️ Instalação e Execução

Siga os passos abaixo para rodar o projeto localmente.

### 1. Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- [Node.js](https://nodejs.org/) (versão recomendada: LTS)
- [Git](https://git-scm.com/)

Verifique as versões instaladas:

```bash
node -v
npm -v
git --version
```

### 2. Clonar o Repositório

Clone este repositório para a sua máquina local:

```bash
git clone https://github.com/LilianMS/BookShelf.git
```

### 3. Instalar as Dependências

Entre na pasta do projeto e instale as dependências:

```bash
cd BookShelf
npm install
```

### 4. Executar o Projeto

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

A aplicação estará disponível em:

👉 <http://localhost:3000>
